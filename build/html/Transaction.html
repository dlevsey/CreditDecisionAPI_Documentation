Transaction#

Endpoints:#

  • POST Transaction/GenerateTransactionDetails

  • POST Transaction/CreateNewIssuance

  • GET Transaction/GetTAStatusTransactions

  • POST Transaction/GenerateTransactions

POST Transaction/GenerateTransactionDetails#

Description:#

  • Allows you to generate transaction details.

  • Only completed transactions can be generated.

  • Pending transaction will not be generated.

Request Body:#

controlTransactionId integer Required

Example Request:#

curl --silent --location '{{baseurl}}/api/v1/Transaction/GenerateTransactionDetails' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <YOUR TOKEN>' \
--data '{
    "controlTransactionId": 2121
}'

POST Transaction/CreateNewIssuance#

Description:#

Allows user to create a new issuance.

Request Body:#

customerIssuerId integer Required
Your System Issuer Id and not SOLO Issuer Id.
processDate string Required
Date transaction was performed. "yyyy-mm-dd (Year-Month-Day)".
chargeAmount integer Required
Amount will charge the issuer for this transaction.
certificateType string Required
Book = Book Shares Physical = Physical Shares
securityType.securitySeriesCode string Optional
A = Series A. B = Series B. C = Series C.
comments string Required
Notes or comments about this transaction.
createdByEmail string Required
Email of person who created Shareholder.
createdByFirstName string Required
First name of person who is creating Shareholder.
createdByLastName string Required
Last name of person who is creating Shareholder
customerIssuerId integer Required
Your System Issuer Id and not SOLO Issuer Id.
restrictionCode string Required
N = No Restriction R = Rule 144 B = Blue Skey A = Afficiliate/Control S = Regulation S L = Lock-Up Agreement O = Other
taxLots string Optional
Represents a unique purchase of a security. Requires taxLots.date
taxLots.date string Required
Represents date of security purchase. ISO 8601 format = standardized date-time representation. e.g., "yyyy-mm-dd" (Year-Month-Day), "2023-12-21T20:36:27.089Z", etc.
taxLots.shares integer Optional
Represents number of shares purchased.
taxLot.pricePerShare integer Optional
Price per share.
taxLot.totalCostBasis integer Optional
Represents original value of investment.

GET Transaction/GetTAStatusTransactions#

Description:#

Allows you to get detailed information about TA Status Transactions.

Query Parameters:#

customerIssuerId integer Required
Your System Issuer Id and not SOLO Issuer Id.
filterTransactionType string Required
Filter transactions by type.
transactionStatus string Required
All , Pending,Rejected, Completed,Canceled
orderBy string Required
Possible values: Ascending, OR Descending.
sortBy string Required
SortName = sorts by last and first name. Name = sorts by Shareholder name. Shares = sort by shares amount.
start integer Required
e.g., 0, 10, 20.
numberOfRows integer Required
Generates a specific number of rows.

Example Request:#

curl --silent --location '{{baseurl}}/api/v1/Transaction/GetTAStatusTransactions?CustomerIssuerId=9999&FilterTransactionType&Start=1&Status=All&Start=0&NumberOfRows=2&SortBy=completeddate&OrderBy=Ascending' \

POST Transaction/GenerateTransactions#

Description:#

Allows you to generate transactions.

Request Body:#

customerIssuerId integer Required
Your System Issuer Id and not SOLO Issuer Id.
filterDateBy string Required
e.g., Effective Date.
startDate string Required
Start date for report. (Defaults to today.) ISO 8601 format = standardized date-time representation. e.g., "yyyy-mm-dd" (Year-Month-Day), "2023-12-21T20:36:27.089Z", etc.
endDate string Required
End date for report. ISO 8601 format = standardized date-time representation. e.g., "yyyy-mm-dd" (Year-Month-Day), "2023-12-21T20:36:27.089Z", etc.
transactionTypeCode string Required
Empty = All Transactions. N = New Issuances. T = Transfers. C = Conversions. R = Retirements. F = Forward Split. V = Reverse Split. D = Dividends.
isPDF bool Required
Generates a PDF. true = generates a downloadable pdf. false = does not generate a downloadable pdf.

Example Request:#

curl --silent --location '{{baseurl}}/api/v1/Transaction/GenerateTransactions' \
    --header 'Content-Type: application/json' \
    --header 'Authorization: Bearer <YOUR TOKEN>' \
    --data '{
    "customerIssuerId": "9999",
    "filterDateBy": "EffectiveDate",
    "startDate": "2020-07-01",
    "endDate": "2020-07-01",
    "transactionTypeCode": "",
    "isPDF": true
    }'
