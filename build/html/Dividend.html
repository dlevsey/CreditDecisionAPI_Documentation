Credit Decision#

Allows you to generate a dividend report for a given dividend.

Endpoints:#

  • GET Dividend/GetList

  • POST Dividend/GenerateDividendDetails

GET Dividend/GetList#

Description:#

Retrieves a list of dividends.

Query Parameters:#

customerIssuerId integer Required
Your System Issuer Id and not SOLO Issuer Id.

Example Request:#

curl --silent --location '{{baseurl}}/api/v1/Dividend/GetList?customerIssuerId=1127' \
    --header 'Authorization: Bearer <YOUR TOKEN>'

POST Dividend/GenerateDividendDetails#

Description:#

Generates a detailed report for a given dividend.

Request Body:#

dividendId integer Required
SOLO System Dividend/Control Id
isPDF bool Required
Generates a PDF. true = generates a downloadable pdf. false = does not generate a downloadable pdf.

Example Request:#

curl --silent --location '{{baseurl}}/api/v1/dividend/generatedividenddetails' \
 --header 'Content-Type: application/json' \
 --header 'Authorization: Bearer <YOUR TOKEN>' \
 --data '{
     "dividendId":4059,
     "isPDF":true
 }'
