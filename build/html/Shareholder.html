Shareholder#

Resource and endpoints for Shareholder related operations.

Endpoints:#

  • POST Shareholder/CreateOrUpdate

  • GET Shareholder/Details

  • GET Shareholder/GetHoldingSummary

POST Shareholder/CreateOrUpdate#

Possible Actions:#

  • Create a new Shareholder, or

  • Update an existing Shareholder.

Request Body:#

createdByEmail string Required
Email of person who created Shareholder.
createdByFirstName string Required
First name of person who is creating Shareholder.
createdByLastName string Required
Last name of person who is creating Shareholder
customerShareholderId string Optional
Id of the Shareholder.
name string Required
First and last name of Shareholder.
name2 string Optional
First and last name of second person on account. E.g., "Emily Johnson."
name3 string Optional
First and last name of third person on account. E.g., "Orion Maddox."
sortName string Required
Sorted first and last name of Shareholder. i.e., “last name, first name”.
accountType string Optional
Account type for Sharheolder.
primaryPayeeName string Optional
Name of Primary Payee.
primaryTaxId string Required
Tax Id for Primary Shareholder.
primaryTaxIdType string Required
Type of primary tax ID. Possible values: SSN, EIN, or SIN.
secondaryPayeeName string Optional
Name of Secondary Payee on account.
secondaryTaxId string Optional
Tax Id for Secondary Payee.
secondaryTaxIdType string Optional
Type of secondary tax ID. Possible values: SSN, EIN, or SIN.
mailingAddress string Optional
Mailing address of Shareholder.
primaryAddress string Required
Primary address of Shareholder.

Primary (or Mailing) Address Child Attributes:#

address1 string Required
Street address for Primary Shareholder.
address2 string Optional
Street address for Primary Shareholder.
city string Required
City of primary address.
state string Required
State of primary address.
postalCode string Required
Postal code of primary address.
country2Code string Required
E.g., US, CA, etc. Complete list of all country ISO codes.
createdByEmail string Required
Email of person who created Shareholder.
phone string Required
Phone number of Shareholder.

curl --silent --location 'https://sandboxapi.transfersolo.com/api/v1/Shareholder/CreateOrUpdate' \

GET Shareholder/Details#

Description:#

Retrieves detailed information about specified Shareholder:

Query Parameters:#

customerShareholderId string Required
Id of the Shareholder.

Example Request:#

curl --silent --location --request GET 'https://sandboxapi.transfersolo.com/api/v1/Shareholder/Details?CustomerShareholderId=T5914714658&SystemShareholderId=0' \
--header 'grant_type: client_credentials' \
--header 'client_id: {{clentId}}' \
--header 'client_secret: {{clientsecret}}' \
--header 'Content-Type: application/x-www-form-urlencoded' \
--header 'Type: text/plain' \
--header 'Authorization: Bearer <YOUR TOKEN>' \
--data-urlencode 'grant_type=client_credentials' \
--data-urlencode 'client_id={{clentId}}' \
--data-urlencode 'client_secret={{clientsecret}}'

GET Shareholder/GetHoldingSummary#

Description:#

Retrieves summary of holdings for specified Shareholder.

Query Parameters:#

customerIssuerId integer Required
Your System Issuer Id and not SOLO Issuer Id.
customerShareholderId string Required
Id of the Shareholder.

Example Request:#

curl --silent --location 'https://sandboxapi.transfersolo.com/api/v1/Shareholder/GetHoldingSummary?customerShareholderId=111144&customerIssuerId=1128' \
    --header 'grant_type: client_credentials' \
    --header 'client_id: {{clentId}}' \
    --header 'client_secret: {{clientsecret}}' \
    --header 'Authorization: Bearer <YOUR TOKEN>'
