GET Report/GetOwnershipPercentage#

Description:#

Allows you retrieve ownership details.

Query Parameters:#

customerIssuerId integer Required
Your System Issuer Id and not SOLO Issuer Id.
cutoff string Required
Cutoff date for report. (Defaults to today.) ISO 8601 format = standardized date-time representation. e.g., "yyyy-mm-dd" (Year-Month-Day), "2023-12-21T20:36:27.089Z", etc.
sortBy string Optional
SortName = sorts by last and first name. Name = sorts by Shareholder name. Shares = sort by shares amount.
orderBy string Optional
Possible values: Ascending, OR Descending.
filterSecurityType string Optional
Has two child attributes: securityTypeCode and securitySeriesCode.
filterSecurityType.securityTypeCode string Optional
Blank = All Securities. C = Common Securities. P = Preferred Securities.
filterSecurityType.securitySeriesCode string Optional
Blank = No series specified. A = Series A. B = Series B. C = Series C.

Note

Refer to the Expanded Attributes and Options page for more info on securityType options.

Example Request:#

curl --silent --location --request GET '{{baseurl}}/api/v1/Report/GetOwnershipPercentage?customerIssuerId=1128&cutoff=2023-07-12&filterSecurityType=null&filterSecurityType.securityTypeCode=null&filterSecurityType.securitySeriesCode=null&sortBy=Name&orderBy=Ascending' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <YOUR-TOKEN>'\
--data '{
    "CustomerIssuerId":1128,
    "Cutoff":"2023-07-12",
    "FilterSecurityType":{
        "SecurityTypeCode":"",
        "SecuritySeriesCode":""
    },
    "SortBy":"Name",
    "OrderBy":"Ascending"
}'

POST Report/GenerateShareholderList#

Description:#

Generates detailed Shareholder List Reports.

Request Body:#

customerIssuerId integer Required
Your System Issuer Id and not SOLO Issuer Id.
cutoff string Optional
Cutoff date for report. (Defaults to today.) ISO 8601 format = standardized date-time representation. e.g., "yyyy-mm-dd" (Year-Month-Day), "2023-12-21T20:36:27.089Z", etc.
isPDF bool Optional
Generates a PDF. true = generates a downloadable pdf. false = does not generate a downloadable pdf.
customerShareholderId string Optional
Id of the Shareholder.
showCertificateDetails bool Optional
Will show details of certificate. true = shows details. false = does not show details.
showAddressInfo bool Optional
Shows address info for each Shareholder. true = shows address info. false = does not show address info
showContactInfo bool Optional
Shows contact info for each Shareholder. true = shows contact info. false = does not show contact info.
showTaxId string Optional
true = shows tax id. false = does not show tax id.
hideCanceledDateIfGreaterThanCutoff bool Optional
true = hides canceled date(s) that occur after cutoff. false = shows canceled date(s) that occur after cutoff.
filterSecurityType string Optional
Has two child attributes: securityTypeCode and securitySeriesCode.
filterSecurityType.securityTypeCode string Optional
Blank = All Securities. C = Common Securities. P = Preferred Securities.
filterSecurityType.securitySeriesCode string Optional
Blank = No series specified. A = Series A. B = Series B. C = Series C.

Note

Refer to the Expanded Attributes and Options page for more info on securityType options.

Prefix string Optional
certificateInfoStatus string Required
All = Outstanding and Canceled Shares. Outstanding = Outstanding Shares. Canceled = Canceled Shares.
restrictionStatus string Required
All = Restricted and Non-Restricted. Restricted = Restricted Shares. NonRestricted = Non-Restricted Shares.
addressStatus string Required
All = Good and Bad Addresses. GoodAddress = Good Address. BadAddress = Bad Address.
includeTaxLots bool Optional
Note: Only used for excel report(s).

            "SecurityTypeCode":"",
            "SecuritySeriesCode":""
        },
        "Prefix":null,
        "IncludeTaxLots":false,
        "certificateInfoStatus": "All",
        "restrictionStatus": "All",

    }'
